## 📅 工作流程

每个任务的执行流程：

1. **任务启动**: 使用 `$golang-pro` 进行开发
2. **代码实现**: 完成具体功能开发
3. **代码审查**: 使用 `$code-reviewer` 检查代码质量
4. **测试验证**: 使用 `$test-automator` 进行自动化测试
5. **集成测试**: 验证与现有功能的兼容性
6. **文档更新**: 更新相关文档
7. **任务完成**: 验收标准检查

- 使用 Go 1.25.8
